# 🧪 Ease Commerce UI Automation using TestCafe

This project contains UI automation tests built using **TestCafe**.

---

## 🚀 Prerequisites

Make sure you have the following installed:

* Node.js (v14 or higher recommended)
* npm (comes with Node.js)
* Google Chrome browser

---

## 📦 Installation

Clone the repository & navigate into project folder:

git clone https://github.com/imManishKumar/TestCafe-EaseCommerce.git

Install all required dependencies:

```bash
npm install
```

---

## ▶️ Running Tests

### ✅ Run all tests

```bash
npx testcafe chrome tests/
```

---

### ✅ Run a specific test file

```bash
npx testcafe chrome .\tests\<fileName>.js
```

> Replace `<fileName>` with your actual test file name.
